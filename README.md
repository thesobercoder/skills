# Skills

Custom [Claude Code](https://claude.com/claude-code) skills.

## Skills

### composio

Universal service aggregator — gateway to 500+ external apps via Composio MCP.

Recommended, install globally for Claude Code:

```
npx skills add thesobercoder/skills -a claude-code -s composio -g -y
```

Install only in the current project:

```
npx skills add thesobercoder/skills -a claude-code -s composio -y
```

### lan-proxy

Expose local services over LAN via Caddy reverse proxy and ufw firewall.

Recommended, install globally for Claude Code:

```
npx skills add thesobercoder/skills -a claude-code -s lan-proxy -g -y
```

Install only in the current project:

```
npx skills add thesobercoder/skills -a claude-code -s lan-proxy -y
```
